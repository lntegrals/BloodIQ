# BloodLens

A Flask-based web application for blood analysis.

## Project Structure
```
blood-iq/
└── bloodlens/
    ├── app.py
    ├── templates/
    │   ├── index.html
    │   └── results.html
    ├── static/
    │   └── style.css
    ├── utils/
    │   └── prompt_builder.py
    ├── requirements.txt
    └── README.md
```

## Setup
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```
